# 8 Puzzle — React Implementation

An interactive implementation of the classic **8-Puzzle Problem** built with React + Vite.

The application demonstrates puzzle mechanics along with the **Manhattan Distance heuristic**, displayed live as you make moves.

For a small Web App like this, React is absolutely unnecessary however I was learning React and I figured this will be a good way to practice :) 

Eventually in the future once I add an actual backend (Python for logic) and API (FastAPI) calls and add all the other minigames and make it a fullstack web app, React will come in handy.

![Screenshot](./screenshot.png)

## Features

- Solvable random shuffling    
- Manhattan distance heuristic display  
- Simple (Minecraft themed ) UI
